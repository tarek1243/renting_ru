Renting.ru

Start everything with one command — from the repo root

1
Start Docker services

PostgreSQL, Redis, MinIO (file storage) and Mailpit (email preview) run in containers.

docker-compose up -d
2
Run migrations + seed (first time only)

Creates all DB tables then loads demo cars, drivers, users, coupons and content.

pnpm --filter @renting/api migrate:deploy
pnpm --filter @renting/api seed
3
Start everything API Web Admin

Turborepo runs all three apps in parallel in a single terminal window.

pnpm dev

ⓘ Run this from the repo root folder. All logs stream together, colour-coded per app.
